# 🔤 DFA & NFA Visualizer

A Python GUI application that converts Regular Expressions into NFA and DFA, and visualizes them using Graphviz.

---

## ✨ Features

- Convert Regular Expressions into **NFA**
- Convert NFA into **DFA**
- Graphical visualization of automata
- Simple GUI built with Tkinter
- Instant preview of generated automata

---

## 🖥️ Interface

- Input field for Regular Expression
- Buttons to generate:
  - NFA
  - DFA
- Display area for the generated automaton graph
